# Java Encapsulation Example

This program demonstrates **Encapsulation** in Java using private variables, getters, setters, and constructors.

## Concepts Covered

- Data Hiding
- Private Data Members
- Getter and Setter Methods
- Constructors
- Object Creation

## Class Used

**Employee**
- Stores employee details like ID, Name, Department, and Age.
- Accesses data using getter and setter methods.
